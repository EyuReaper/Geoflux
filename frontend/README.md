# GeoFlux Frontend Operations

Frontend stack: React 19 + Vite + TypeScript.

## Prerequisites

- Node.js 20+
- npm 10+
- Backend API running on `http://localhost:4000`

## Install

```bash
npm install
```

## Run (Development)

```bash
npm run dev
```

Default local URL: `http://localhost:5173`

## Validate

Run tests:
```bash
npm run test
```

Build for production:
```bash
npm run build
```

Lint:
```bash
npm run lint
```

## Current Notes

- Tests are passing.
- TypeScript production build currently fails due to store/type mismatches that need stabilization before release.
